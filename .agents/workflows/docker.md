# Docker Servislerini Başlat

// turbo-all

1. Docker servislerini başlat:
```bash
docker compose up -d
```

## Adresler
- **PostgreSQL**: localhost:5432
- **Redis**: localhost:6379
- **Seq Logs**: http://localhost:5341
- **RabbitMQ**: http://localhost:15672 (guest/guest)
